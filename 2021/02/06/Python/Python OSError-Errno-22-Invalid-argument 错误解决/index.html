Python OSError: [Errno 22] Invalid argument 错误解决

发表于2021-02-06|更新于2021-07-24|Python错误
|字数总计:188|阅读时长:1分钟|阅读量:

问题描述

在爬取视频数据并保存到本地时报错：OSError: [Errno 22] Invalid argument

解决思路

在爬取其他视频时并没有报此错误，对比后发现是视频名称的问题：在调用open函数时，文件路径中含有字符|，导致报错。

问题解决

在爬取文件名时，将符号|替换掉。这里直接将其删去。

至此问题得以解决。
